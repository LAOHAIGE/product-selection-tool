选品决策台便携版

使用方法：
1. 先完整解压 zip，不要在压缩包里直接双击。
2. 双击“启动选品工具.cmd”。
3. 浏览器会自动打开工具页面。
4. 在页面的“AI 配置”区域填写自己的 DeepSeek Key，点击“保存 AI 配置”，再点击“测试 Key”。

如果双击后没有打开页面：
- 重新双击“启动选品工具.cmd”，启动窗口现在会保持打开并显示错误。
- 查看 logs\launcher.log，把里面的错误发给开发者。

说明：
- 工具数据会保存在 app\data 目录。
- 公开分享版默认不包含发送者的 DeepSeek Key；每个人需要在自己的电脑上配置自己的 Key。
- 启动器会优先使用 runtime\node.exe；如果这个文件存在，对方电脑通常不需要再安装 Node.js。
- 如果启动时提示“未找到 Node.js”，说明这个包没有带 runtime\node.exe，或者被杀毒软件拦截了。请让发送者重新打包，或安装 Node.js LTS。
- 这个版本适用于 Windows 电脑。
